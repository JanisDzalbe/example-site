# Test Automation Practice Site

A collection of web pages designed for learning and practicing web automation testing techniques.

## Purpose

This site provides hands-on examples and interactive exercises for testing various web automation scenarios, including:

- **Element Locators** - Practice identifying elements using different strategies (IDs, classes, nested structures)
- **User Actions** - Test interactions with forms, buttons, inputs, checkboxes, dropdowns, and date pickers
- **Synchronization** - Handle timing scenarios with delays, loading animations, and dynamic content
- **Alerts & Popups** - Work with JavaScript alerts, confirms, prompts, and page navigation dialogs
- **Form Workflows** - Complete multi-step forms with validation and error handling

## Structure

- **`examples/`** - Demo pages showcasing specific automation concepts
- **`tasks/`** - Interactive practice exercises for hands-on learning

## Live Site

Access the site at: **https://janisdzalbe.github.io/example-site/**

## Usage

Navigate through the examples and tasks using the menu on each page. Each page is designed to present different challenges commonly encountered in web automation testing.
